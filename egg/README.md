# vue-admin-serve egg.js服务

### 开始

```bash
$ cd egg/ #先进入到这个目录
$ npm i
$ npm run dev
$ open http://127.0.0.1:7001
```

### 具体文档

[查看具体开发文档](https://hangjob.github.io/docs/#/vue-bag-admin/)

### Deploy

```bash
$ npm start
$ npm stop
```
